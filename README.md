## Installation

```bash
$ npm install
```

```bash
# file
$ .env
```

## # Swagger API docs

- http://localhost:3000/api

## Running with Docker / Docker compose

```bash
# development
$ docker-compose up --build
```

```bash
# development - Detached mode: Run containers in the background
$ docker-compose up -d --build
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test
